# Zleepz-Project
# 🧠 Team Zleeps — AI Classroom Sleep & Attention Detector

## 📌 Overview
Team Zleeps developed an AI-powered classroom attention monitoring system that detects whether a student is attentive, distracted, or sleepy using webcam input and AI analysis.

The system analyzes student behavior in real time and classifies attention levels into:
- AWAKE
- PARTIALLY ATTENTIVE
- SLEEPY

It also provides:
- Attention percentage
- Alert signals
- Smart notifications
- Loud buzzer alerts for sleep detection

# 🧠 How the System Works

### Workflow:
Webcam Input → AI Analysis → Attention Detection → Alert System → Output Display

The webcam captures student frames and sends them to Gemini AI for analysis. The AI then classifies attention level and returns structured outputs.

---

# 📊 Attention Classification System

## 1. AWAKE
- Student is focused and looking forward
- Attention Level: 80–100%
- Alert: NO_ALERT

## 2. PARTIALLY ATTENTIVE
- Student is distracted or looking away
- Attention Level: 40–79%
- Alert: SOFT_WARNING

## 3. SLEEPY
- Eyes closed or head down
- Attention Level: 0–39%
- Alert: BEEP_ALERT

---

# 🔊 Alert System
The system triggers:
- Loud buzzer alerts for sleepy students
- Soft warnings for distracted students
- Positive feedback for attentive students

Example Messages:
- “Wake up detected 💀”
- “Focus mode needed”
- “Locked in. Keep going 🔥”

---

# 🧩 AI Prompt Logic

The AI system follows these rules:
- Eyes closed + head down → SLEEPY
- Looking away → PARTIALLY ATTENTIVE
- Looking forward → AWAKE

The AI returns:
- State
- Attention percentage
- Alert signal
- Short feedback message
